개발 기간 : 2023.03.27~ 2023.06.21
- JDBC Templete 사용
- 배포 AWS 사용

- 문자인증 로그인 구현 
- 당근마켓 주요 서비스 기능 
- AWS 서버 구축, 무 중단 배포
